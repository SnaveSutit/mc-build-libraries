# SnaveSutit's MC-Build Libraries

A collection of tools and libraries for MC-Build Data Pack projects.

## Libraries

- [Execute](execute/README.md) - A template that adds block-based syntax sugar for each `execute` sub-command.
- [Interactions](interactions/README.md) - Run commands when a player right / left clicks an interaction entity
- [Score](score/README.md) - A shorthand template for `scoreboard` commands.
- [Sequence](sequence/README.md) - An intuitive interface for creating a sequence of scheduled functions.
- [While](while/README.md) - A template that adds runtime sync & async `while`, `until`, `do-while`, and `do-until` loops.
